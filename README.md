---

## Deploy to GitHub Pages

1. Create a GitHub repo named `portfolio`
2. Upload all files (`index.html`, `style.css`, `script.js`, `README.md`)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Live in ~60 seconds at: `https://yourusername.github.io/portfolio`
